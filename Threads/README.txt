EdurekaTimer- Timer app that uses threads. Can be paused and continues from last stopping point.

EdurekaTriviaGame: Consists of client+server. Server connected on a port sends short questions to client. Client connects to same port and is able to answer. When client sends "stop" the game ends.

CheckersStockTaking: Consists of client+server. Client sends details  to server which is saved to a database

Cell - Server + client. Client sends in cellphone info, server takes info and saves to a text file
